# First Steps

Your initial orientation to the Osgiliath Enterprise ecosystem.

## Step 1: Access Your Workspace

Log into the enterprise portal using your credentials. You'll find:

- Project dashboards
- Team channels
- Resource directories

## Step 2: Clone the Repositories

```bash
git clone git@github-kb:OsgiliathEnterprise/<project>.cd <project>
npm install
```

## Step 3: Set Up Your Environment

Follow the [Environment Setup Guide](/docs/quickstart/environment-setup) to configure your local development environment.

## Step 4: Join the Channels

- **Discord**: General discussion and help
- **Slack**: Team-specific communication
- **GitHub**: Issue tracking and code review

## Next Steps

Once you're set up, explore the [Architecture Overview](/docs/architecture/overview) to understand how everything fits together.
